0. **Resolver Pontos de Atenção:**

1. **Setup de Vitrine Moderna:**
   - Se ainda não houver runner, configure `Vitest` (mais rápido se em ecossistema vite) ou `Jest`. Em backends rust, python ou go, ative os runners locais equivalentes (cargo test, pytest).
2. **Defesa de Regras Core (Integração de Alto Valor):**

3. **Defesa Continuous (Pipeline YAML):**
   - Caso não exista automação de deploy, crie o pipeline básico de checagem.
   - Crie `.github/workflows/ci.yml`. Adicione nele `npm install`, seguido das tasks vitais `npm run lint`, `npm run build`, e `npm run test` (ou o que estiver no `package.json`).
4. **Validation:** Execute o `run_command` rodando o teste recém-criado para comprovar que está verde e testando um fluxo válido.

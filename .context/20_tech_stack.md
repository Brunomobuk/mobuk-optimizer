# 🔧 Tech Stack

## Core

| Tecnologia | Uso |
|------------|-----|
| Markdown (`.md`) | Linguagem principal — todos os prompts, skills, workflows e rules |
| YAML Frontmatter | Metadados de workflows e skills (name, description) |
| Mermaid | Diagramas de fluxo e arquitetura |

## Ecossistema

| Tecnologia | Uso |
|------------|-----|
| Gemini / Antigravity | IA que interpreta e executa os agents |
| `.agents/` convention | Estrutura reconhecida automaticamente pelo sistema de agentes |
| Slash commands (`/`) | Interface de entrada do usuário para workflows |
